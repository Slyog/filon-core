# FILON

A visual knowledge system built with **Next.js**, **TypeScript**, and **ReactFlow**.  
FILON visualizes ideas as interactive nodes and connections — structure meets creativity.

### Stack
Next.js · React 18 · TailwindCSS · Prisma · ReactFlow  

### Development
```bash
npm install
npx prisma generate
npm run dev
```
